# Awesome SMS APIs

A curated list of SMS API providers for developers.

## Global Providers

| Provider | Coverage | Pricing | Features |
|----------|----------|---------|----------|
| Twilio | 180+ countries | Pay-per-use | Programmable SMS, MMS |
| Vonage | 200+ countries | Pay-per-use | SMS API, Verify API |
| MessageBird | 190+ countries | Pay-per-use | Flow Builder, Conversations |
| Plivo | 190+ countries | Pay-per-use | Bulk SMS, Long codes |
| Telnyx | 180+ countries | Pay-per-use | Carrier-grade, Low latency |

## Regional Providers

### North America
- Bandwidth - Enterprise SMS/MMS
- Sinch - Carrier connections

### Europe
- Infobip - Omnichannel platform
- CM.com - Business messaging

### Asia
- Alibaba Cloud SMS - China coverage
- Exotel - India focused

## Open Source

- [Jasmin](https://github.com/jookies/jasmin) - SMS gateway
- [PlaySMS](https://playsms.org/) - Web-based SMS portal
